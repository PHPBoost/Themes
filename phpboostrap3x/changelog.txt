
Installation
    Une fois le thème installé, vous devez actualiser le cache tpl (/administration/outils/cache) de façon à prendre en compte les éléments du dossier default.

Specifications
    Navigation
        le deploiement des sous-menus se fait au clic.
    Les réglages de nombre d'item par lignes sont désactivés, remplacés par col-viewport-x (x=4 en général => 3 colonnes)
    -pbt
        A la manière de primary, warning, danger, etc... une class-pbt à été ajoutée dans phpboostrap-theme.css
    Font Awesome
        Comme la librairie FA est intégrée dans pbt, les icônes Glyphicon ne sont pas activées.
        Le bouton responsive et l'icone dropdown (.caret) des menus navbar/dropdown sont remplacés par des icônes FA.
    Eléments Bootstrap
        Certaines alertes ne pouvant pas être modifiées en html (générées en php), leurs alias sont définis dans phpboostrap-theme.css : danger=error; info=question; et une classe notice aux couleur de la classe btn-default a été créée.
        De même pour la classe alert-link, les classes <alert-x a> ont été personnalisées.
        Les modifications de classes de bootstrap sont définies dans phpboostrap.css (section Reset bootstrap) de manière à ne pas interferrer sur la mise à jour de bootstrap.css et bootstrap-theme.css .
        La classe .input-group-separator a été ajoutée pour permettre 2 input cote à cote dans une classe .input-group
    Fichiers css
        PHPBoost possédant son propre minifier, il n'est pas nécessaire de charger les fichiers.min.css de Bootstrap
    Bootstrap-grid-columns.js
        Permet de mettre tous les items et/ou les sous-catégories d'une catégorie d'une même ligne à la même hauteur (hack pour le float:left qui décale le design si un article est plus long)
    Navbar.js
        Permet de gerer les menus actionslinks en front
